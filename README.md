# 02435 Decision-Making Under Uncertainty – Assignment Part A

This repository contains the code and data for **Part A of the course assignment** for *02435 Decision-Making Under Uncertainty*, Spring 2026, at the Technical University of Denmark (DTU).

## Tech Stack

- **NumPy** – numerical computations
- **pandas** – data handling
- **Gurobi (gurobipy)** – optimization modeling and solving
- **Matplotlib** – data visualization
